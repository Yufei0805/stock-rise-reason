# 股票上涨原因分析工具

基于社区讨论和专业投研数据，使用AI生成简洁的股票上涨原因分析。

## ✨ 功能特点

- 📊 **多数据源整合**：集成5个数据源（3个社区平台 + 2个专业投研）
- 🚀 **批量分析**：支持单个或多个股票同时分析
- 📁 **Excel支持**：支持从Excel文件读取股票列表
- 🤖 **AI分析**：生成100字以内的简洁上涨原因分析
- 📝 **独立运行**：包含所有必要文件，无需依赖其他工具

## 🎯 数据源

### 社区讨论（3个平台）
- 韭研公社
- 东方财富股吧
- 淘股吧

### 专业投研（2个数据源）
- AlphaPai 路演纪要
- AlphaPai 机构点评

## 🚀 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
playwright install chromium
```

### 2. 使用方法

**单个股票**：
```bash
python analyzer.py 英科医疗
python analyzer.py 300677
```

**多个股票**：
```bash
python analyzer.py 英科医疗 沃尔核材
python analyzer.py 300677,002130
```

**从Excel读取**：
```bash
python analyzer.py --excel stocks.xlsx
python analyzer.py --excel "D:\我的股票.xlsx"
```

### 3. 输出结果

脚本会：
1. 爬取所有股票的社区讨论和投研数据
2. 显示收集到的数据
3. 等待AI生成分析（需配合Claude使用）
4. 保存为txt文件

## 📋 Excel格式支持

### 格式1：标准列名

包含"代码"或"名称"关键词的列会被自动识别：

```
| 股票代码 | 股票名称 | 其他列... |
| 300677  | 英科医疗 | ...      |
```

### 格式2：A列元组字符串

A列包含元组格式，自动提取代码和名称：

```
| A列                                    |
| ★('000525.SZ', '红太阳', '农化制品')    |
| ('000818.SZ', '航锦科技', '化工')      |
```

自动处理：去除.SZ/.SH后缀，提取股票代码和名称
