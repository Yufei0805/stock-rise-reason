# 股票上涨原因分析工具

基于社区讨论和专业投研数据，使用AI生成简洁的股票上涨原因分析。

## ✨ 功能特点

- 📊 **多数据源整合**：集成5个数据源（3个社区平台 + 2个专业投研）
- 🚀 **批量分析**：每5只股票一批，大幅提升效率
- 📁 **Excel支持**：支持从Excel文件读取股票列表
- 🤖 **AI分析**：生成100字以内的简洁上涨原因分析
- 🔄 **智能去重**：自动跳过最近7天已分析的股票
- ⚙️ **配置驱动**：通过config.yaml灵活配置所有参数
- 📝 **独立运行**：包含所有必要文件，无需依赖其他工具

## 🎯 数据源

### 社区讨论（3个平台）
- 韭研公社
- 东方财富股吧
- 淘股吧

### 专业投研（2个数据源）
- AlphaPai 路演纪要
- AlphaPai 机构点评

## 🚀 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
playwright install chromium
```

### 2. 使用方法

**单个股票**：
```bash
python analyzer.py 英科医疗
python analyzer.py 300677
```

**多个股票**：
```bash
python analyzer.py 英科医疗 沃尔核材
python analyzer.py 300677,002130
```

**从Excel读取**：
```bash
python analyzer.py --excel stocks.xlsx
python analyzer.py --excel "D:\我的股票.xlsx"
```

### 3. 输出结果

脚本会自动完成以下步骤：
1. 爬取所有股票的社区讨论和投研数据（每5只一批）
2. 显示批量分析提示词
3. 自动生成txt模板文件（等待Claude填写）
4. Claude分析后，结果保存到txt文件

**输出文件**：
- 位置：`D:\Claude专用文件夹\`（可在config.yaml配置）
- 格式：`股票上涨原因分析_YYYYMMDD.txt`
- 内容：每只股票100字以内的上涨原因分析

## 📋 Excel格式支持

### 格式1：标准列名

包含"代码"或"名称"关键词的列会被自动识别：

```
| 股票代码 | 股票名称 | 其他列... |
| 300677  | 英科医疗 | ...      |
```

### 格式2：A列元组字符串

A列包含元组格式，自动提取代码和名称：

```
| A列                                    |
| ★('000525.SZ', '红太阳', '农化制品')    |
| ('000818.SZ', '航锦科技', '化工')      |
```

自动处理：去除.SZ/.SH后缀，提取股票代码和名称

## ⚙️ 配置说明

### 配置文件：config.yaml

工具通过 `config.yaml` 进行配置，无需修改代码：

```yaml
# 分析设置
analysis:
  batch_size: 5  # 每批分析的股票数量（建议5-10）
  output_length: 100  # 每只股票分析字数限制

# 去重设置
deduplication:
  enabled: true  # 是否启用去重
  days: 7  # 跳过最近N天已分析的股票
```

**批量分析优势**：
- 10只股票：从10次对话减少到2次
- 节省时间和API成本
- 避免重复分析

### AlphaPai配置（可选）

如需获取专业投研数据，需配置AlphaPai API。如未配置，只显示社区讨论数据。
