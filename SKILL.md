# 股票上涨原因分析 Skill

## 功能定位

基于社区讨论和专业投研数据，快速分析股票近期上涨原因。

**独立运行**：本skill包含所有必要文件，无需依赖其他skill。

## 数据源（最近15天）

本skill集成了3个数据源：

### 社区讨论

#### 韭研公社
- URL: `https://www.jiuyangongshe.com/search/new?k={股票名称}`
- 方法: Playwright（浏览器环境）
- 获取数量: 6条
- 时间过滤: 15天内

### 专业投研（最近15天）

#### 路演纪要
- 数据源: AlphaPai API
- 类型: roadShow, roadShow_ir
- 获取数量: 5条
- 包含: 标题、时间、内容摘要

#### 机构点评
- 数据源: AlphaPai API
- 类型: comment
- 获取数量: 5条
- 包含: 标题、时间、机构名称、内容摘要

### 搜索引擎补充

Claude自动搜索雪球、东财、淘股吧等社区的最新讨论（15天内）

## 使用方法

### 方式1：单个股票
```bash
python analyzer.py 英科医疗
python analyzer.py 300677
```

### 方式2：多个股票
```bash
python analyzer.py 英科医疗 沃尔核材
python analyzer.py 300677,002130
python analyzer.py 300677 002130 000708
```

### 方式3：从Excel读取

#### 命令行方式
```bash
python analyzer.py --excel stocks.xlsx
python analyzer.py --excel D:\我的股票.xlsx
python analyzer.py --excel "D:\我的 文件夹\股票列表.xlsx"
```

#### 通过Claude使用（推荐）
直接告诉Claude您的Excel路径：
```
帮我分析 D:\我的股票.xlsx
用这个Excel分析：C:\Users\xxx\stocks.xlsx
```

Claude会自动：
1. 运行analyzer.py读取Excel
2. 爬取所有股票的数据
3. 生成每只股票的上涨原因分析
4. 保存为txt文件

#### Excel格式要求

**格式1：标准列名**（推荐）

自动识别包含以下关键词的列：
- 包含"代码"的列 → 股票代码
- 包含"名称"或"股票"的列 → 股票名称

示例：
```
| 股票代码 | 股票名称 | 其他列... |
| 300677  | 英科医疗 | ...      |

| 证券代码 | 证券名称 | 行业 | 市值 |
| 300677  | 英科医疗 | ... | ... |
```

**格式2：A列元组字符串**

A列包含元组格式的字符串，自动提取代码和名称：

示例：
```
| A列                                    |
| ★('000525.SZ', '红太阳', '农化制品')    |
| ★('000818.SZ', '航锦科技', '化工')      |
| ★('600511.SH', '国药股份', '医药')      |
```

自动处理：
- 提取代码：`000525`（自动去除.SZ/.SH后缀）
- 提取名称：`红太阳`

**位置要求**：
- ✅ 相对路径：`stocks.xlsx`（当前目录）
- ✅ 绝对路径：`D:\任意位置\stocks.xlsx`
- ✅ 带空格路径：需要加引号
- ✅ 任意列数：格式1需要有"代码"或"名称"列，格式2只需A列

## 工作流程

### 第一步：爬取数据
脚本自动爬取韭研公社和AlphaPai投研数据，每5只股票一批（约15-20秒/批）

### 第二步：Claude搜索
Claude自动搜索雪球、东财、淘股吧等社区的最新讨论（15天内）

### 第三步：批量分析
- 显示批量分析提示词（包含所有股票数据）
- 自动生成txt模板文件

### 第四步：Claude分析
Claude基于提示词，批量生成每只股票100字以内的上涨原因分析

### 第五步：自动保存
- 分析结果自动保存到txt文件
- 文件位置：`D:\Claude专用文件夹\股票上涨原因分析_YYYYMMDD.txt`
- 同时保存分析历史，7天内自动去重

## 常见问题

### Q1: 如何提高分析准确性？
A: 提供更多股票的历史数据，让Claude有更多参考信息。

### Q2: 支持多少个股票同时分析？
A: 理论上无限制，但建议一次不超过10个，避免执行时间过长。

### Q3: 如果某个数据源失败怎么办？
A: 每个数据源独立错误处理，单个失败不影响其他数据源。

### Q4: 可以分析下跌原因吗？
A: 当前版本专注上涨原因，可修改提示词分析下跌原因。

### Q5: 输出文件保存在哪里？
A: 由Claude决定保存位置，通常在`D:\Claude专用文件夹\`
